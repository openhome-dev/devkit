# OpenHome Devkit

Hardware files for the OpenHome Devkit — 3D print files, CAD, and community enclosure designs.

## Official Designs

| Version | Folder | Status |
|---------|--------|--------|
| V1.0 | [`designs/v1.0/`](designs/v1.0/) | Current |

## Files

- `.3mf` — Print-ready project files (recommended starting point)
- `STL Files/` — Individual parts for slicing
- `Step Files/` — STEP files for editing in Fusion 360, FreeCAD, SolidWorks, etc.

## Community Designs

Community remixes and custom enclosures live in [`community/`](community/). Have a modified design? Open a PR.

## Contributing

1. Fork this repo
2. Add your design to `community/your-design-name/`
3. Include a `README.md` describing what you changed and what printer/settings you used
4. Open a PR
